# The Green Toolbox — An Interactive Courtroom Drama

A fun, self-contained web experience that presents the courtroom dialogue as a staged, scene-by-scene reading.

## How to run

Just open `index.html` in any modern browser. No build step. No dependencies.

## Controls

- **Click the bench / "BEGIN THE TRIAL"** — starts the show.
- **Click anywhere, press `Space`, `Enter`, or `→`** — advance to the next page.
- **`←`** — go back a page.
- **`Esc`** — restart from the cover.
- **Autoplay** — let the scenes advance on their own.

## Features

- Three-act structure with animated act banners.
- Color-coded speakers (Judge, Lawyer, Son/Daniel, Murderer, Bailiff, Clerk).
- Stage directions set off from dialogue in italic.
- Ceiling fans spinning in the background.
- Gavel-crack screen flash at the climax of Act I.
- Responsive layout for desktop and mobile.

## Files

- `index.html` — page structure.
- `styles.css` — courtroom styling.
- `script.js` — story data + page logic.
